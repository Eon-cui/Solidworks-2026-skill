#### 2.1.1 安全加固

| # | 问题 | 严重度 | 修复 |
|---|------|:------:|------|
| 1 | 文件写入无路径限制 | **高** | `_validate_output_path(path)` — 环境变量 `SW_OUTPUT_DIR`（默认 `./output/`），`os.path.realpath` + 🆕 `os.path.normcase` + `startswith`，否则 `PermissionError` |
| 2 | `async def` 内 `time.sleep()`（8 处） | **高** | → `await asyncio.sleep()` |
| 3 | `EditRebuild3` 无操作 bug（L1498） | **中** | → `_com_attr(doc, 'EditRebuild3')` 实际调用 |
| 4 | 函数体内 import（5 处） | **低** | → 文件顶部 |
| 5 | 裸 `except Exception: pass`（13 处） | **中** | → 最低 `traceback.print_exc()` |
| 6 | `_byref_int` 无 COM 回退崩 | **中** | → 返回有 `.value` 属性的包装对象 |
| 7 | 未使用导入 + `VIEW_MAP` | **低** | → 删除 |
| 8 | JSON 无大小限制 | **中** | → `MAX_JSON_SIZE = 10MB` 检查 |
| 9 | STEP 文件无大小限制 | **低** | → 100MB 上限 |

**`_validate_output_path` 实现（🆕 加 `normcase`）：**
```python
import os as _os

_SW_OUTPUT_DIR = _os.environ.get("SW_OUTPUT_DIR", _os.path.join(_os.getcwd(), "output"))

def _validate_output_path(path: str) -> str:
    real = _os.path.normcase(_os.path.realpath(_os.path.abspath(path)))
    allowed = _os.path.normcase(_os.path.realpath(_SW_OUTPUT_DIR))
    if not real.startswith(allowed + _os.sep) and real != allowed:
        raise PermissionError(
            f"Output path '{path}' resolves to '{real}' which is outside "
            f"allowed directory '{allowed}'. Set SW_OUTPUT_DIR env var to change."
        )
    _os.makedirs(_os.path.dirname(real), exist_ok=True)
    return real
```
