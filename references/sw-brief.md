# SW Brief — 建模前结构化需求提取

> 写 `with SW()` 之前先填 brief。不是给用户填的表格，是你自己理清思路的草稿。

## 为什么

直接跳代码 → 参数散落、坐标系搞混、验证目标不明确 → 返工。brief 把"要建什么"和"怎么验"写死在建模前。与 `part-modeling.md` 分工：后者 = 代码怎么写，本文档 = 写代码前怎么想。

## Brief 格式

建模前在代码注释或单独笔记里写清以下字段。不确定的值标 `(假设)`。

```text
SW Brief:
- 零件名: <PascalCase, 与 SLDPRT 文件名一致>
- 任务类型: 新零件 / 修改 / 装配 / 导出
- 模板: <gb_part.prtdot | gb_asm.asmdot | 自定义路径>
- 坐标系: <Y-up (GB 默认) | Z-up (ANSI)> — TOP 平面法向
- COM 会话: with SW() / 底层 sw_part 函数(米接口)
- 整体尺寸: <长×宽×高 mm>
- 功能特征:
  - <特征1>: <尺寸+公差意图>
  - <特征2>: ...
- 配合/接口 (如有):
  - <接口名>: <PCD/孔径/面选择策略/配合类型>
- 输出路径: <SLDPRT+STEP 成对, 目录>
- 验证目标:
  - check_faces: {底盘: 1, 中心孔: 1, 4×安装孔: 1}
  - verify_step: holes=[(r_mm, count, name), ...], max_circle=<R_mm>
  - verify_assembly_poses (装配时): [(label, R, t_mm), ...]
- 假设: <所有推断值显式列出>
```

## 澄清策略

只在缺信息导致**无法建模**时才问。以下场景不追问直接假设：

| 场景 | 默认假设 |
|------|----------|
| 未指定倒角半径 | 1-3mm 合理范围 |
| 未指定孔类型 | M3 通孔 φ3.4, M4 通孔 φ4.5, M5 通孔 φ5.5 |
| 未指定壁厚 | 小型塑料壳 2-3mm |
| 未指定坐标系 | GB 模板 Y-up |
| 未指定输出目录 | 与脚本同目录 |

**必须问的场景：** 安全关键件、配合面未定义、载荷件无尺寸、合规件无标准号。

## 示例：一个支架

用户说："做个 L 形支架，底板 80×60×6，竖板高 40 厚 6，底板四个 M4 安装孔 PCD 36@45°，竖板两个 φ10 轴承孔间距 30。"

Agent brief：

```text
SW Brief:
- 零件名: Bracket
- 任务类型: 新零件
- 模板: gb_part.prtdot
- 坐标系: Y-up (GB 默认) — TOP=底板顶面, FRONT=竖板前面
- COM 会话: with SW()
- 整体尺寸: 80×60×46 mm (宽×深×高=底板+竖板)
- 功能特征:
  - 底板: 80×60×6, 圆角 R4 (假设)
  - 竖板: 80×40×6, 底板后边缘对齐
  - 4×M4 安装孔: PCD36 @45°, φ3.4 通孔, 底板
  - 2×φ10 轴承孔: 间距30, 竖板, 通孔
- 配合/接口:
  - 底板底面 → 安装面 (平面贴合)
  - 4×M4 孔 → M4 螺栓 (PCD36 @45°)
  - 2×φ10 孔 → 轴承外圈
- 输出路径: D:\out\Bracket.SLDPRT + .STEP
- 验证目标:
  - check_faces: {底板拉伸: 1, 竖板拉伸: 1, 底板圆角: 1, 4×M4孔: 1, 2×φ10孔: 1}
  - verify_step: holes=[(1.7, 4, 'φ3.4×4'), (5.0, 2, 'φ10×2')], max_circle=None
- 假设: 圆角 R4, 倒角未指定→不做, 轴承孔公差未指定→FDM 孔+0.2
```

## 与代码的同步

brief 中的验证目标直接复制进 `verify_step()` / `check_faces()` 调用。brief 中的 LAYOUT 直接复制进 `add_component_posed()` 的 LAYOUT 列表。**brief=代码的单事实来源。**

改设计时先改 brief，再改代码，最后重跑验证。brief 和代码不同步 → 技术债。

## 就绪判据

brief 可以开始建模当且仅当：
- 模板路径已知（或可用 glob 查找）
- 坐标系方向已声明
- 每个特征的形状+尺寸+位置可写出
- 验证目标已用具体数字表达
- 所有假设已显式记录
